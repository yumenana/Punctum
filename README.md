# Punctum

> *"That accident which pricks me — but also bruises me, is poignant to me."*
> — Roland Barthes, *Camera Lucida*

Punctum 是一个关于**认知架构**的探索性研究项目。

项目名取自 Roland Barthes 在《明室》(*Camera Lucida*) 中的概念——照片里那个"刺穿你"的细节, 那个无法忽视的"**这个**"。这个词同时具有拉丁语"点/刺"的词根含义, 与项目的图像背景和"指针"概念三重共鸣。

---

## 核心问题

现有的视觉-语言模型 (如 Grounding DINO, LLaVA, SAM) 把"指向某个具体区域"当作一个**需要从大数据中学出来的任务**。

Punctum 的问题是: 如果我们把**"指示" (deixis)** 做成架构里的一等公民——一个显式的结构性算子——而不是让它从训练中涌现, 会怎样?

---

## 思想源头

项目背后的原始 AGI 构想, 见 [`docs/Genesis.md`](docs/Genesis.md)。

这套思路在大语言模型大规模出现之前已独立提出, 包含:

- 认知本质是统计学 + 多模态感知接地是必要条件
- 元概念引导的递归解构学习
- 从"指示"出发, 自底向上建立概念体系
- 理解最终应内化为直觉, 而非每次重新推理
